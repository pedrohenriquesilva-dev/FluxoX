# Sobre o Projeto
O FluxoX nasceu da necessidade de transformar uma planilha de controle financeiro em um sistema moderno, inteligente e prático. Em vez de fórmulas manuais e abas no Excel, o objetivo é ter uma aplicação web completa que:

Registra receitas e despesas de forma rápida e organizada
Separa automaticamente movimentações eletrônicas de dinheiro físico (espécie)
Calcula economia real mensal e compara com uma meta desejada
Exibe um painel anual completo com visão de janeiro a dezembro
Controla valores guardados em diferentes contas e bancos
Realiza conferência entre saldo teórico e saldo real

# O Problema que Resolve
Quem controla as finanças em planilhas sabe a dor:

Fórmulas que quebram quando você move uma célula
Sem filtros práticos por categoria ou forma de pagamento
Difícil visualizar se você está acima ou abaixo da meta
Não dá pra usar bem no celular
Não separa o que é dinheiro físico do que é eletrônico
Não mostra se o saldo guardado bate com o que deveria ter

# Como Rodar Localmente

Pré-requisitos

Node.js 18+
npm

# Clone o repositório
git clone https://github.com/pedrohenriquesilva-dev/FluxoX.git

# Entre na pasta
cd FluxoX

# Instale as dependências
npm install

# Rode o servidor de desenvolvimento
npm run dev

Acesse: http://localhost:5173
Build para produção
bashnpm run build
npm run preview
