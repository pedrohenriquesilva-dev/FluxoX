# Release Notes v2.0 — FluxoX

Este pacote inclui a versão final de entrega para o Dia 40, com a documentação completa e o histórico de mudanças.

## Principais novidades

- Documentação final do projeto em **README.md**.
- Histórico completo de mudanças em **CHANGELOG.md**.
- Consolidação de funcionalidades de controle financeiro, exportação e PWA.
- Arquivo de release notes incluso para uso na página de releases do GitHub.

## Funcionalidades destacadas

- **Dashboard completo** com cards de métricas e gráficos interativos.
- **Transações de receitas e despesas** com CRUD, filtros e ordenação.
- **Exportação CSV** de dados filtrados para despesas, receitas e resumo anual.
- **Resumo mensal** formatado para copiar/compartilhar.
- **PWA offline** com install banner e cache de assets.
- **Visão anual** com tabela consolidada e saldo acumulado.

## Observações

- O pacote `dia-40.zip` contém a pasta `dia-40/` com:
  - `README.md`
  - `CHANGELOG.md`
  - `RELEASE-NOTES-v2.md`
- Todos os arquivos foram gerados a partir do estado atual do projeto.
- Este arquivo serve como resumo para a release do GitHub.
